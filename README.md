## 🧮 Формулы конвертации

### Линейная регрессия (R² = 0.999)

```
Lichess Rapid → FIDE:   FIDE = 1.0005 · Lichess_Rapid − 248
Chess.com Rapid → FIDE: FIDE = 0.9748 · Chess_Rapid − 185
```
