## 🎯 Введение

Рейтинг FIDE — официальная система оценки силы игры в шахматы. Однако большинство современных шахматистов играют онлайн на Lichess и Chess.com. 

**Цель работы:** построить математическую модель для оценки рейтинга FIDE на основе онлайн-рейтингов.

## 🧮 Формулы конвертации

### Линейная регрессия (R² ≈ 0.999)

**Lichess Rapid → FIDE Standard:**
```
FIDE = 1.0005 × Lichess_Rapid − 247.62
```
Упрощённо: FIDE ≈ Lichess_Rapid − 248

**Chess.com Rapid → FIDE Standard:**
```
FIDE = 0.9748 × Chess_Rapid − 185.04
```
Упрощённо: FIDE ≈ Chess_Rapid × 0.97 − 185
