## 🎯 Введение

Рейтинг FIDE — официальная система оценки силы игры в шахматы, признанная во всём мире. Однако большинство современных шахматистов играют преимущественно онлайн на платформах Lichess и Chess.com, где используют собственные рейтинговые системы. 

**Цель работы:** построить математическую модель для оценки рейтинга FIDE на основе рейтингов Lichess и Chess.com.

**Задачи:**
1. Собрать данные о рейтингах игроков на всех трёх платформах
2. Проанализировать распределения и корреляции
3. Построить регрессионные модели конвертации
4. Реализовать веб-калькулятор с открытой методологией

## 📈 Методология сбора данных

### Источники данных

```
Lichess API → https://lichess.org/api/user/{username}
Chess.com API → https://api.chess.com/pub/player/{username}/stats
FIDE API → https://ratings.fide.com/profile/{fide_id}
```

### Выборка

- **Титулованные игроки** (GM, IM, FM, WGM, WIM) — для привязки к FIDE
- **Активные пользователи** (>100 игр в рейтинговом пуле)
- **Сбалансированное распределение** по диапазонам рейтингов (800–3000)

### Процесс сбора

1. Получение списка титулованных игроков через Lichess API
2. Поиск соответствующих профилей на Chess.com
3. Получение FIDE ID через поиск или известные базы
4. Сбор рейтингов: Standard/Rapid/Blitz на каждой платформе
5. Сохранение в структурированный датасет (JSON)

## 📐 Анализ данных

### Распределение рейтингов

_Графики распределения будут построены после сбора данных:_

- **Lichess Rapid** — типичное распределение игроков
- **Chess.com Rapid** — распределение игроков
- **FIDE Standard** — распределение официальных рейтингов

### Корреляционная матрица

Коэффициенты корреляции Пирсона между:
- Lichess Rapid ↔ Chess.com Rapid
- Lichess Rapid ↔ FIDE Standard
- Chess.com Rapid ↔ FIDE Standard
- Blitz ↔ Rapid (внутри платформы)
- Разные контроли времени

### Регрессионный анализ

**Линейная регрессия:**
```
FIDE = β₀ + β₁ · OnlineRating + ε
```

**Полиномиальная регрессия (2-й степени):**
```
FIDE = β₀ + β₁ · OnlineRating + β₂ · OnlineRating² + ε
```

**Кусочно-линейная интерполяция:**
Набор референсных точек с линейной интерполяцией между ними.

### Метрики качества
- R² (коэффициент детерминации)
- RMSE (среднеквадратичная ошибка)
- MAE (средняя абсолютная ошибка)
- Доверительные интервалы (95%)

## 🧮 Формулы конвертации

_Будут уточнены после полного сбора данных_

### Предварительные оценки

**Lichess Rapid → FIDE Standard:**
```
FIDE ≈ LichessRapid × 0.85 + 150
```

**Chess.com Rapid → FIDE Standard:**
```
FIDE ≈ ChessRapid × 0.82 + 180
```

### Интервалы уверенности

| Уровень | Отклонение | Условие |
|---------|-----------|---------|

| 🟡 Средний | ±100 | 100–300 пунктов от референса |
| 🔴 Низкий | ±150 | Более 300 пунктов от референса |

## ✅ Результаты и выводы

_Будут дополнены после завершения сбора данных_

Ожидаемые результаты:
1. Зависимость между онлайн и офлайн рейтингами нелинейна
2. Lichess Rapid выше FIDE Standard на ~100–300 пунктов (в зависимости от диапазона)
3. Chess.com Rapid → FIDE: разница уменьшается с ростом рейтинга
4. Blitz-рейтинги менее надёжны для предсказания FIDE Standard
